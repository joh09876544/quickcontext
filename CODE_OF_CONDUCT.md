# Code of Conduct

## Our Standard

This project aims to be respectful, practical, and welcoming to contributors.

Participants are expected to:

- communicate respectfully
- assume good intent
- accept technical feedback without hostility
- keep disagreements focused on code and design decisions
- avoid harassment, abuse, threats, discrimination, and personal attacks

## Expected Behavior

- Be clear and professional in issues, pull requests, and discussions.
- Give feedback that is specific and useful.
- Respect different skill levels and backgrounds.
- Keep discussions focused on improving the project.

## Unacceptable Behavior

- Harassment or intimidation
- Personal attacks or insults
- Discriminatory or hateful language
- Bad-faith disruption of discussions or reviews
- Sharing private or sensitive information without permission

## Enforcement

Project maintainers may edit, lock, hide, or remove content that violates these standards and may restrict participation when necessary.

## Reporting

If you see behavior that should be reviewed, open a private security-style report if sensitive, or contact the maintainers through the repository support path described in `SUPPORT.md`.
